  1. Добавить поле поиска в хедер страницы(должно присутствовать на каждой странице приложения)
  2. При вводе от 3 символов и более должен появляется выпадающий список с подсказками, по списку можно перемещаться с помощью стрелок верх в низ, либо выбрать нужный пункт мышью(функционал UI полностью аналогичен вводу запроса в гугле). Поиск осуществляется только по имени компании, введёная строка может быть как полным именем, так и просто быть частью имени(подстрокой). Если ни один из пунктов не выбран, а нажата клавиша Enter то осуществляется переход на страницу со списком компаний подходящим критерию поиска(ввод в инпуте поиска). Если выбран пункт из списка подсказки(туда выводить названия подходящих компаний) то осуществлять переход на страницу с полным описанием выбранной компании.

    1. на запросы поиска при вводе надо отсылать json с данными для автоподсказки
    2. страница вывода результатов поиска(имена компаний в списке должны быть ссылками на их страницы), отображение если поиск дал 0 результатов
